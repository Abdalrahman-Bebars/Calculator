# Calculator
Calculator is a Web App that uses Node.js to allow to submit the inputs of the page to calculate the result of the inputs given.

#### Please use the following for the operators:

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## What I Used
- HTML
- Node.js
- Express.js
- body-parser NPM
